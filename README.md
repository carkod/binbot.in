# Binbot.in

## Overview

Binbot.in is the brochure website for the Binbot customers, built with React, Vite, and Tailwind CSS. It provides a UI prospective customers information to get in touch with BINBOT LTD.

## Related Repositories

- **binbot**: Backend API and services
- **binbot-notebooks**: Data science and research notebooks
- **binbot-charts**: Charting library and visualization tools
- **pybinbot**: Python SDK and utilities
- **binquant**: Quantitative trading algorithms

## Getting Started

### Prerequisites

- Node.js (v20). Check the nvmrc file
- npm

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:5173` (or as configured).

### Build

Build for production:

```bash
npm run build
```

## Development Notes

- Uses Radix UI for accessible components
- Tailwind CSS for utility-first styling
- Vite for fast builds and HMR
- TypeScript for static typing
- Scripts for dev, build, preview, and type checking
